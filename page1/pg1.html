Course: Programming

What You'll Learn in the Programming Course:

In this course, you will learn the basics of programming. You will understand what code is and how to write it. You will study things like variables, numbers, text and how to make the computer do different actions. You will also learn how to find and fix mistakes in your code. Step by step, you will learn to think like a programmer and write simple programs. You will learn popular coding languages like Python, HTML, CSS, JavaScript and more. This course is great if you want to start learning how to build websites, games, or other computer tools.

Course: Game development

What You'll Learn in the Game development Course:

In this course, you will learn how to create your own games from the beginning. You will understand how games are made and how to make them fun to play. You will also learn how to plan and design your game ideas. You will study important topics like game design, characters, levels, points and how to make a game look good. You will also learn how to test your game, find mistakes (bugs), and fix them. Step by step, you will get better and start thinking like a game developer. This course is great if you enjoy games and want to learn how to build your own for fun or in the future as a job.

Course: Robo-Tech

What You'll Learn in the Robo-Tech Course:

In this course, you will learn how robots work and how to build simple ones yourself. You will understand the basics of robotics, electronics. You will study topics like sensors, motors, lights, and how robots can see and understand the world. You will also learn how to give instructions to a robot using code. You will use tools and languages like Arduino, Python, Scratch, roblox studio or block-based coding, and learn about real technology. This course is great if you enjoy building things, solving problems, and learning how machines can help people in the real world.
